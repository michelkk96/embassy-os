# Changing Your Password

To change your StartOS master password, go to **System > Change Password** in the main UI and enter a new password. You do not need to enter your current password — being signed in is sufficient.

> [!WARNING]
> Existing backups remain encrypted with the password that was set when they were created. Changing your password does not re-encrypt them, so keep your old password until you no longer need those backups.
