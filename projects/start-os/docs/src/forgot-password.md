# Forgot Password

If you have lost or forgotten your StartOS master password, there are two ways to reset it.

## From a signed-in device

If any of your devices still has a signed-in StartOS session, go to **System > Change Password** on that device and set a new password. Your current password is not required — see [Change Password](change-password.md).

## By reinstalling

If no device is still signed in, you can reset your password by reinstalling StartOS while preserving your data.

1. Download and flash the latest version of StartOS, using the appropriate [installation guide](installing-startos.md) for your hardware.
1. Visit http://start.local.
1. When selecting your data drive, select **Keep my data**.
1. Create a new password and complete setup. All your previous addresses and data will be preserved.
